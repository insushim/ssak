# 시스템 테스트 체크리스트

이 문서는 AI 글쓰기 도우미 시스템의 모든 기능을 테스트하기 위한 체크리스트입니다.

## 사전 준비

- [ ] Node.js v18 이상 설치 확인
- [ ] npm 의존성 설치 완료 (`npm install`)
- [ ] Firebase 프로젝트 생성 완료
- [ ] `.env` 파일 설정 완료
- [ ] Firestore Security Rules 설정 완료
- [ ] 슈퍼 관리자 계정 생성 완료
